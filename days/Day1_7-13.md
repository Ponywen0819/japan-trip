# Day 1｜7/13（一）落地直奔臨空城 Outlet → 傍晚進心齋橋
> 中午落地，正好接上 outlet 營業時間。下飛機帶行李一站到臨空城，寄物逛街，傍晚再進大阪，沒有趕班機壓力、從容很多。

| 時段 | 行程 |
|------|------|
| 11:25 | 關西機場落地。入境＋提行李約 1 小時（約 12:30 出關） |
| 12:40 | 搭 南海電鐵 → 臨空城站（りんくうタウン），1 站約 6 分 |
| 12:50 | 站外有人寄物處放行李（一件約 ¥300-500），走聯絡空橋（穿過 SEACLE）約 6 分到 outlet |
| 13:00–16:30 | 臨空城 Rinku Premium Outlets 採買。逾 250 家品牌，精品、日系服飾、運動戶外、雜貨。開到 20:00，時間充裕可慢慢逛 |
| 中午 | outlet 內餐廳或美食廣場午餐 |
| 16:30 | 取回行李，臨空城站搭 南海電鐵 → 南海難波站（約 50 分） |
| 傍晚 | 南海難波站出站沿商店街步行 10-15 分到心齋橋飯店放行李（行李重可搭計程車） |
| 晚上 | 道頓堀 晚餐＋夜逛。視體力順逛唐吉訶德（24h）或心齋橋筋 |

- **午餐：** 臨空城 outlet 內
- **晚餐：** 道頓堀。大牧場（高 CP 燒肉、神戶牛吃到飽）或千房大阪燒，清爽選道頓堀今井烏龍麵
- **住宿：** 心齋橋哈頓飯店（ハートンホテル心斎橋，已訂，7/13–7/15 共 2 晚，入住 15:00 起）
  - 地址：〒542-0086 大阪府大阪市中央區西心齋橋 1-5-24（1 Chome-5-24 Nishishinsaibashi）
  - 電話：+81-6-6251-3711
  - Email：heartonhotel.4vk7u81u4xz30rt@htlpartner.trip.com
  - 就在心齋橋，逛街、道頓堀晚餐都步行可達。
- **行李提醒：** 第一天行李通常還沒塞滿，outlet 買的東西塞得下；但取出後要一路拖進大阪，買多會變重。到臨空城站務必先寄物再逛。
- **退稅提醒：** outlet 買的「一般物品」（服飾、鞋、包）可使用、只要 7/18 帶出境即可；「消耗品」（藥妝、食品）辦免稅須封袋到出境前不可拆，會封一週。藥妝零食類建議改在大阪市區（唐吉訶德）買就好。
