# Day 4｜7/16（四）金澤｜聚焦兼六園，慢遊不趕
> 不貪多。這天以兼六園為核心慢慢逛，周邊順路的當「有力氣再加」，累了就回飯店休息，保留品質。

| 時段 | 行程 |
|------|------|
| 上午 | 睡飽出發 → 近江町市場 吃海鮮丼當早午餐（從片町步行約 18-20 分或搭公車） |
| 中午～下午 | 兼六園（本日主角）。七月看新綠、霞之池與徽軫石燈籠、翠瀧瀑布，慢慢逛慢慢拍，不趕時間 |
| 下午 | 還有力氣才加：緊鄰的 金澤城公園（就在兼六園隔壁，順走） |
| 傍晚 | 回片町休息，或順遊 東茶屋街 吃金箔甜點（看體力決定） |
| 晚上 | 片町一帶晚餐 |

- **午餐：** 近江町市場（豐水產招牌丼附 15 種食材加金箔、或一市や能登和牛喉黑魚）
- **晚餐：** 片町。金澤在地味，金沢おでん（關東煮）或壽司
- **住宿：** MYSTAYS 金澤片町飯店（ホテルマイステイズ金沢片町，同前晚，不換宿）
- **重點：** 兼六園是這天唯一一定要走的，其餘全是選配。金澤城就在兼六園旁，順腳走幾乎不額外花力氣；東茶屋街較遠（離片町約 25-30 分），累了就跳過、留到 Day 5 研討會前後有空再去。
- **兼六園小提醒：** 七月初夏有燕子花、新綠與雨後青苔之美。早上 8 點前人少、清晨最清幽（園內 7:00 開）。禁用自拍棒和腳架。
