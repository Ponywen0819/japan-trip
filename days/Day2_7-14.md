# Day 2｜7/14（二）大阪｜睡飽再出發，下午大阪城
> 前一天累了、早上爬不起來很正常。這天不排早上行程，睡飽自然醒，重點放下午的大阪城（天守閣＋御座船），傍晚回市區購物。

| 時段 | 行程 |
|------|------|
| 上午 | 睡飽自然醒、悠閒早午餐。不排行程 |
| 13:30–14:30 | 抵達 大阪城公園（最晚 14:30 到，確保趕上天守閣末入館 16:30） |
| 下午 | 大阪城天守閣（定點）。內部博物館＋登頂展望，抓 1.5-2 小時。亮點：5 樓「夏之陣」動態模型、2 樓租戰國武將頭盔拍照（¥500） |
| 約 16:30 | 出天守閣，旁邊 御座船 碼頭搭金色遊船遊護城河（約 20 分，末班約 16:50 前；成人 ¥1,500）。從水面仰望石垣與天守閣 |
| 傍晚 | 公園黃昏拍城景，搭車回難波/心齋橋 |
| 晚上 | 心齋橋、難波購物＋晚餐 |

- **午餐：** 起床後悠閒早午餐，或大阪城周邊
- **晚餐：** 心齋橋/難波，選擇最多
- **住宿：** 心齋橋哈頓飯店（ハートンホテル心斎橋，同 Day 1，不換宿）
- **時間關鍵：** 天守閣最後入館 16:30、御座船末班約 16:50（3-10 月開到 16:50）。先逛天守閣（時間瓶頸）再搭船最保險，別倒過來。中午前出門、13:30 前到就很充裕。
- **想加碼：** 若你起得比預期早、或大阪城逛得快，傍晚回程路上可順加梅田藍天大廈看夜景，或新世界通天閣，但非必要、別勉強。
