# 大阪 × 金澤 六天五夜行程表

**旅遊主軸：** 關西專心玩大阪，金澤兩晚從容，最後配合研討會。京都、奈良留待下次。

## 固定條件（已訂）
- 7/13 中午 11:25 落地關西，住大阪（7/13、7/14）
- 7/15 早上大阪退房 → 金澤，住金澤（7/15、7/16）
- 7/17 下午研討會，晚上回大阪住
- 7/18 下午 15:20 關西機場飛回台灣

**唯一定點：** 大阪城（Day 1）

---

## Day 1｜7/13（一）落地直奔臨空城 Outlet → 傍晚進心齋橋
> 中午落地，正好接上 outlet 營業時間。下飛機帶行李一站到臨空城，寄物逛街，傍晚再進大阪，沒有趕班機壓力、從容很多。

| 時段 | 行程 |
|------|------|
| 11:25 | 關西機場落地。入境＋提行李約 1 小時（約 12:30 出關） |
| 12:40 | 搭 南海電鐵 → 臨空城站（りんくうタウン），1 站約 6 分 |
| 12:50 | 站外有人寄物處放行李（一件約 ¥300-500），走聯絡空橋（穿過 SEACLE）約 6 分到 outlet |
| 13:00–16:30 | 臨空城 Rinku Premium Outlets 採買。逾 250 家品牌，精品、日系服飾、運動戶外、雜貨。開到 20:00，時間充裕可慢慢逛 |
| 中午 | outlet 內餐廳或美食廣場午餐 |
| 16:30 | 取回行李，臨空城站搭 南海電鐵 → 南海難波站（約 50 分） |
| 傍晚 | 南海難波站出站沿商店街步行 10-15 分到心齋橋飯店放行李（行李重可搭計程車） |
| 晚上 | 道頓堀 晚餐＋夜逛。視體力順逛唐吉訶德（24h）或心齋橋筋 |

- **午餐：** 臨空城 outlet 內
- **晚餐：** 道頓堀。大牧場（高 CP 燒肉、神戶牛吃到飽）或千房大阪燒，清爽選道頓堀今井烏龍麵
- **住宿：** 心齋橋哈頓飯店（ハートンホテル心斎橋，已訂，7/13–7/15 共 2 晚，入住 15:00 起）。地址：大阪市中央區西心齋橋 1-5-24；電話 +81-6-6251-3711。就在心齋橋，逛街、道頓堀晚餐步行可達。
- **行李提醒：** 第一天行李通常還沒塞滿，outlet 買的東西塞得下；但取出後要一路拖進大阪，買多會變重。到臨空城站務必先寄物再逛。
- **退稅提醒：** outlet 買的「一般物品」（服飾、鞋、包）可使用、只要 7/18 帶出境即可；「消耗品」（藥妝、食品）辦免稅須封袋到出境前不可拆，會封一週。藥妝零食類建議改在大阪市區（唐吉訶德）買就好。

## Day 2｜7/14（二）大阪｜睡飽再出發，下午大阪城
> 前一天累了、早上爬不起來很正常。這天不排早上行程，睡飽自然醒，重點放下午的大阪城（天守閣＋御座船），傍晚回市區購物。

| 時段 | 行程 |
|------|------|
| 上午 | 睡飽自然醒、悠閒早午餐。不排行程 |
| 13:30–14:30 | 抵達 大阪城公園（最晚 14:30 到，確保趕上天守閣末入館 16:30） |
| 下午 | 大阪城天守閣（定點）。內部博物館＋登頂展望，抓 1.5-2 小時。亮點：5 樓「夏之陣」動態模型、2 樓租戰國武將頭盔拍照（¥500） |
| 約 16:30 | 出天守閣，旁邊 御座船 碼頭搭金色遊船遊護城河（約 20 分，末班約 16:50 前；成人 ¥1,500）。從水面仰望石垣與天守閣 |
| 傍晚 | 公園黃昏拍城景，搭車回難波/心齋橋 |
| 晚上 | 心齋橋、難波購物＋晚餐 |

- **午餐：** 起床後悠閒早午餐，或大阪城周邊
- **晚餐：** 心齋橋/難波，選擇最多
- **住宿：** 心齋橋哈頓飯店（同 Day 1，不換宿）
- **時間關鍵：** 天守閣最後入館 16:30、御座船末班約 16:50（3-10 月開到 16:50）。先逛天守閣（時間瓶頸）再搭船最保險，別倒過來。中午前出門、13:30 前到就很充裕。
- **想加碼：** 若你起得比預期早、或大阪城逛得快，傍晚回程路上可順加梅田藍天大廈看夜景，或新世界通天閣，但非必要、別勉強。

## Day 3｜7/15（三）大阪 →（京都中繼）→ 金澤｜訪京大朋友＋鴨川
> 京都剛好在大阪往金澤的雷鳥號線上，順道下車當中繼站。行李寄京都站置物櫃，輕裝去京大找朋友、鴨川散步拍照，再續往金澤。不貪景點、輕鬆為主。

| 時段 | 行程 |
|------|------|
| 中午 | 大阪睡到自然醒，吃午餐，回心齋橋哈頓飯店退房（須 12:00 前退房） |
| 下午 | 大阪 → 京都站（JR 約 30 分或新快速）。行李寄京都站置物櫃（大櫃 ¥700、可過夜；滿了用 B1 Crosta 人工寄物 ¥700/件） |
| 下午 | 輕裝前往 京都大學（左京區吉田本町）找朋友。校園環境與台大相似、世界百大名校，逛起來別有感觸 |
| 傍晚 | 與朋友沿 鴨川三角洲（出町柳）散步拍照——就在京大旁，兩河匯流、有烏龜跳石，當地人放空的河岸，氛圍悠閒。也是柯南與平次相遇的同一條鴨川 |
| 傍晚 | 回京都站取行李，搭 特急雷鳥號 → 敦賀轉北陸新幹線 → 金澤（約 2 小時餘） |
| 晚上 | 抵金澤，搭車/計程車到片町飯店放行李，片町一帶晚餐 |

- **午餐：** 大阪出發前吃
- **晚餐：** 金澤片町。金澤在地味，金沢おでん（關東煮）、壽司或居酒屋
- **住宿：** MYSTAYS 金澤片町飯店（ホテルマイステイズ金沢片町，已訂，7/15–7/17 共 2 晚）。地址：〒920-0981 石川縣金澤市片町 1 丁目 10-18。
- **行李關鍵：** 一定要在京都站把行李寄好再去京大，別拖著行李跑。京都站約 3000 個寄物櫃，過夜寄放沒問題。
- **時間抓法：** 這天是「移動＋輕度京都」，別貪心。柯南其他場景（清水寺、鞍馬寺、南禪寺）都太遠、移動日不划算，已捨棄。若跟朋友碰完還有力氣，六角堂、五条大橋在市中心可順加，否則直接回站搭車。
- **訂票：** 雷鳥號建議提前劃位；發車時間配合你離開京都的時間彈性選。注意班次別太晚，免得到金澤過深。

## Day 4｜7/16（四）金澤｜聚焦兼六園，慢遊不趕
> 不貪多。這天以兼六園為核心慢慢逛，周邊順路的當「有力氣再加」，累了就回飯店休息，保留品質。

| 時段 | 行程 |
|------|------|
| 上午 | 睡飽出發 → 近江町市場 吃海鮮丼當早午餐（從片町步行約 18-20 分或搭公車） |
| 中午～下午 | 兼六園（本日主角）。七月看新綠、霞之池與徽軫石燈籠、翠瀧瀑布，慢慢逛慢慢拍，不趕時間 |
| 下午 | 還有力氣才加：緊鄰的 金澤城公園（就在兼六園隔壁，順走） |
| 傍晚 | 回片町休息，或順遊 東茶屋街 吃金箔甜點（看體力決定） |
| 晚上 | 片町一帶晚餐 |

- **午餐：** 近江町市場（豐水產招牌丼附 15 種食材加金箔、或一市や能登和牛喉黑魚）
- **晚餐：** 片町。金澤在地味，金沢おでん（關東煮）或壽司
- **住宿：** 金澤片町（同前晚，不換宿）
- **重點：** 兼六園是這天唯一一定要走的，其餘全是選配。金澤城就在兼六園旁，順腳走幾乎不額外花力氣；東茶屋街較遠（離片町約 25-30 分），累了就跳過、留到 Day 5 研討會前後有空再去。
- **兼六園小提醒：** 七月初夏有燕子花、新綠與雨後青苔之美。早上 8 點前人少、清晨最清幽（園內 7:00 開）。禁用自拍棒和腳架。

## Day 5｜7/17（五）金澤研討會 → 晚上回大阪
> 重點日，移動與行李要先安排好。

| 時段 | 行程 |
|------|------|
| 早上 | 金澤飯店退房，行李寄放飯店或車站 |
| 上午 | 研討會前的空檔，市區機動補點（近江町市場買伴手禮、咖啡） |
| 下午 | 研討會 |
| 傍晚 | 研討會結束 → 取行李 → 金澤站搭車回大阪（約 2.5 小時） |
| 晚上 | 抵達大阪，飯店入住。視抵達時間簡單吃宵夜/晚餐 |

- **注意：** 研討會若較晚結束，回到大阪可能已深夜，先有心理準備。建議當天一早就把行李寄放在金澤站，不必繞回飯店。
- **住宿：** 新大阪微笑飯店（スマイルホテル新大阪，已訂，7/17–7/18）。地址：大阪市澱川區西中島 6-1-11；電話 +81-6-6309-7755。就在新大阪站旁，金澤搭新幹線直達新大阪站下車即到。
